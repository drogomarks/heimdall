Heimdall: Email & Apps API Tool

What is Heimdall?

Heimdall is simple web application that makes calls to the Rackspace Email & Apps API on your behalf. The idea is to be able to leverage the features of the API for complex or bulk tasks in a simple and easy manner for support.

The API Documention for which this follows can be found here: http://api-wiki.apps.rackspace.com

Current Status

The application is pretty much in alpha, needs a lot of work, mobile site is broken (although not sure if people would actually use this on mobile). These are the functions that can be performed currently:

GET

  • Get a list of all mailbox users (ex or rs)
  • Get data about a specfic mailbox user (ex or rs)

PUT

  • Set the value of visibleInExchangeGAL to true or false for ALL users.
  • Change a users password (ex or rs)

What's Next?

The real driving force of this tool is going to be users. This is we have the Feature Request page. To really make this something valuable, submit the things that would help make your job easier, no matter how off the wall it sounds or if you think it's not possible.

Regarding feature Roadmap:

- Currently when you make a call the output returns JSON in a new tab (and XML for a one). The plan is to actually inject the input into the same container, on the same page.

- A seperate site optimized for mobile

- A "call constructor" that would allow the user to choose the HTTP_VERB URL and VALUE(S) and make a desired call to the API.

tl;dr we need YOUR feedback!
